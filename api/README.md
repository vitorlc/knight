## Description

Knights CRUD using Nestjs, MongoDB and Docker.

## Installation

```bash
$ npm install
```

## Initialize the database

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
